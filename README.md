## ⚡ Quick Reference Included

### Critical T-Codes
| T-Code | Purpose |
|--------|---------|
| EA00 | Billing Run (Main invoicing) |
| EA05 | Check billing documents |
| EA19 | View out-sorted documents |
| BP | Business Partner |
| ES32 | Display Installation |
| FPE1 | Post FICA document |
| SE16N | Table browser |

### Must-Know Tables
- **ERCH** - Billing document header ⭐
- **DFKKOP** - FICA open items (unpaid bills) ⭐⭐
- **EVER** - Installation / Contract link
- **EVBS** - Premise master data

## 🏆 EY Interview Tips

- ✦ Use STAR format: Situation, Task, Action, Result
- ✦ Back up with numbers - "500K customers", "60% error reduction"
- ✦ Show integration knowledge - IS-U doesn't work in silos
- ✦ Mention both IS-U + ABAP Debug expertise
- ✦ Align with EY values: Quality, Integrity, Teamwork

## 💡 Key Concepts at a Glance

**BMD** → BP → Contract Account → Contract  
**TMD** → Premise → Installation → Device → Register  
**M2C** → Read → Validate → Bill → Print → Pay → Clear
